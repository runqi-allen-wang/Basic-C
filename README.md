# SJTU Data Structure Assignments 2025-2026-02

This repository records my code submissions for the **Data Structure** course at **Shanghai Jiao Tong University** during the **2025-2026-02 semester**.

The original problem statements are available on the SJTU Online Judge:

https://acm.sjtu.edu.cn/OnlineJudge/problem

## Repository Overview

This repository is mainly used for:

- organizing my data structure homework solutions;
- keeping a personal archive of course programming assignments;
- recording implementations of common data structures and algorithms;
- tracking relatively challenging problems separately.

All code files are written primarily in **C++**.

## File Naming Convention

The source files are named in the following format:

```text
HW?_????.cpp
````

where:

* `HW?` denotes the homework number;
* `????` denotes the corresponding problem ID or problem index;
* `.cpp` indicates that the solution is written in C++.

For example:

```text
HW1_1001.cpp
HW2_1234.cpp
HW3_2048.cpp
```

Some relatively difficult or more important problems are marked with `_star`:

```text
HW?_????_star.cpp
```

For example:

```text
HW2_1234_star.cpp
```

The `_star` suffix indicates that the problem is more challenging, more representative, or worth reviewing again.

## How to Compile

Most files can be compiled independently using a standard C++ compiler.

For example, with `g++`:

```bash
g++ HW1_1001.cpp -o main
./main
```

If the program uses C++17 features, compile with:

```bash
g++ -std=c++17 HW1_1001.cpp -o main
./main
```
